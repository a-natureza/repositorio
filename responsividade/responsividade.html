Responsividade em Textos

Não dá para dizer ao certo qual é o melhor editor de código-fonte, mas pesquisas indicam que o VS Code é o favorito da maioria dos programadores. Graças à sua imensa comunidade, existem inúmeras extensões para o VS Code, de forma que podemos ficar confusos ao tentar discernir o que vale a pena, por isso, hoje te apresentamos 6 extensões para o VS code que ajudam na sua produtividade. Antes de mais nada, é importante ressaltar que o VS Code é um software de código aberto da Microsoft, sob a licença MIT, mas antes de publicá-lo, sua proprietária adiciona alguns conteúdos próprios com código fechado. Quem fica desconfortável com isso, pode optar por compilar diretamente o código-fonte do VS Code, ou então, baixar o VSCodium. Conheça 6 extensões para o VS Code Quando você decidir qual linguagem de programação quer usar, busque pelo nome dessa linguagem entre as extensões disponíveis, esta pode ser uma boa forma de encontrar algo útil para auxiliar no seu trabalho. De qualquer forma, é difícil separar o que vale a pena utilizar e é necessário tomar cuidado com armadilhas, então confira algumas extensões para o VS Code que acreditamos que vale a pena dar uma chance. Hungry Delete Com essa extensão, conseguimos apagar várias linhas em branco seguidas com um só comando, o Ctrl + Backspace, dessa forma, você economiza tempo nessa tarefa repetitiva. Auto Rename Tag O Auto Rename Tag é uma extensão para quem trabalha com HTML e quer renomear rapidamente as tags do código-fonte. Ela concatena cada tag inicial com sua respectiva tag final, dessa forma, ao alterar uma, a outra também será modificada automaticamente. Auto Close Tag Mais uma extensão para HTML, o Auto Close Tag é capaz de fechar automaticamente todas as tags que você abre. Better Comments Esta extensão consegue customizar os seus comentários dentro do código-fonte, deixando o seu trabalho mais organizado, bem documentado e auxiliando em serviços colaborativos. Para utilizá-la, basta abrir um comentário e quando for escrever algo que requer atenção, digite um ponto de exclamação no início, que o texto ficará vermelho, se for uma dúvida, a interrogação deixará azul, o asterisco significa que é uma informação importante e o texto ficará verde, já o termo TODO, significa que algo ainda está por fazer e o texto terá a cor laranja. Prettier O Prettier é uma extensão que deixa o seu código com a indentação correta, ele suporta as linguagens JavaScript, TypeScript, Flow, JSX, JSON, CSS, SCSS, Less, HTML, Vue, Angular HANDLEBARS, Ember, Glimmer, GraphQL, Markdown e YAML. Basta instalá-lo e começar a programar, que a extensão automaticamente fará a indentação para você. Live Server Uma extensão muito popular, o Live Server cria um servidor virtual que permite que você observe imediatamente o resultado do seu desenvolvimento web sem precisar montar o servidor de forma manual, muito útil, por exemplo, para quem programa em PHP. Após instalar esta extensão, você terá um novo botão no canto inferior esquerdo chamado “Go Live”, clicando nele, a página do que você está fazendo será aberta no navegador e ela atualizará automaticamente, conforme suas alterações.
